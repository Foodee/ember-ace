# Ember Ace

An Ember component wrapping [Ace editor](https://ace.c9.io).

## Installation

```bash
ember install ember-ace
```

## Usage

```hbs
{{ember-ace lines=10 value=value update=(action 'valueUpdated')}}
```

### Options

value
update
ready

mode
theme
useSoftTabs
tabSize
useWrapMode
highlightActiveLine
showPrintMargin
printMarginColumn
showInvisibles
readOnly

lines
minLines
maxLines

markers
annotations
